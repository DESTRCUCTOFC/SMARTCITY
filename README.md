# City - Smart

Plataforma de uyn aciudad inteligente desarrollada por BeeStudents utilizando tecnologias para FrontEnd, UI/UX y acceso a algunas API's públicas.

## Caracteristicas 
- Login  y Registro de usuarios
- Dashboard Responsivo 
- integracion con Firebase (Pasos Evolutivos)
- Cinsumo de API's Públicas (Clima,Noticias,Calidad del Aire).

## Tecnologia Utilizada (Stack de Desarrollo)
- HMTL5.
- CSS3.
- Bootstrap 5 .
- Javascript.
- Firebase.
